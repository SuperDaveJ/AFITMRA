To be effective in planning the Manufacturing Readiness Assessment, team members from outside the program need to familiarize themselves with the program.

The Program Manager collects and provides the necessary program data and documentation to the team so that it is in place prior to the assessment and ready to review.

After the data and documentation is provided to the Manufacturing Readiness Assessment Team, the team convenes at Program Management Office for briefings. If necessary, a video teleconference or telecom may be substituted for a face-to-face meeting.
